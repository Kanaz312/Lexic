To run our app:

Open two command line windows in toplevel lexic directory

Command line A:<br>
cd lexic-backend<br>
npm install<br>
npm start<br>


Commad line B:<br>
cd lexic-frontend<br>
npm install<br>
npm install react-router-dom --save<br>
npm start<br>
